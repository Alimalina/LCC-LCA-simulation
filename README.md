# LCC-LCA-simulation

The current code describes the analysis of Life cycle cost (LCC) and Life cycle assessment (LCA) for building renovation process. The analysis includes the following stages - A1-A3 (production), B4 (replacement), B6 (operational energy use) and C1-C4 (end of life).
